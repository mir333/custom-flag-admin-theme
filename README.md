# custom-flag-admin-theme

## build instruction
### preconditions
You need to have npm and gulp installed.
### build
1. chcek out the repo
2. run ```npm install```
3. run ```gulp build```

the war file will be produced in dist folder. Copy the file to Liferay deploy folder.

To make ```gulp deploy``` work you need to configure the liferay-theme.json. See Liferay documentation.
